Employee Accommodation HR/Manager Approval Odoo Apps

Accommodation for Employees Odoo App optimizes the process of accommodation requests initiated by employees. It offers customizable access rights, allowing users to designate roles such as HR and Manager for managing employee accommodation. This Odoo app supports the creation of multiple hotels in various locations, enhancing flexibility. Users can generate requests for accommodations for their employees, which are submitted to managers for the approval process. HR and managers can review and approve or reject requests properly. Additionally, this app enables the generation of expense reports for employee accommodation info in PDF format.

Features

Accommodation Access Rights

User can give an access rights as 'HR' and 'Manager' for an employee accommodation.

Create Accommodation for Employee

User can create accommodation request for an employee and submit to the manager and HR for approval.

Approve/Reject Employee Accommodation by HR/Manager

HR/Manager have access to approve or reject the hotels and employee accommodation request.

Create Hotels with Location

User can create multiple hotels with their locations.

Generate Expense of Employee Accommodation

HR can generate an expense of employee accommodation request.

Print Employee Accommodation Report

User can print employee accommodation report in PDF format.
